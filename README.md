# Harpie

## Description

- Symfony 5.4 (BackEnd)
- Twig/VueJs (FrontEnd)
- PostgresSQL 11 (Base de données)


## Download composer dependencies

```bash
composer install
```

## Running Symfony Applications

```bash
symfony server:start
```
